EMT

PHP/SQL Project

Project Goal

The goal of the EMT website/database was to store scenarios that a common active EMT would recieve. I was responsible of handling the creation, deletion, editing, and searching of these tests, and had to allow for print friendly tests that would fit onto a page. We developed using PHP and SQL in combination with a local MAMP server to test the project.

Project Outcome

The data base was completed with 4 months of inception of the project, I not only taught my team web development, in html and php, but learned more PHP, SQL, and gained experience with database management. The project was never put online becuase it was only used on a local server on the client's desktop.

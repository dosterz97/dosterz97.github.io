Generation+1

Wordpress Website

Project Goal

Generation+1 was a unique concept, as I was to link old documents from another website, and represent the content in an attractive modern way. This was my first wordpress.org website, and the goal was to create the largest website I have ever made, with a large database of files, and a large pool of users that would be joining and accessing content on a consistent basis.

Project Outcome

Generation+1 was an awkward finish. I used the Advanced Custom Fields to build the website so that the client would be able to handle and run the website after I was done with development. After the website was mostly done, the client did not follow through with the plan, and left me with a cool business template for any business that offers subscription based services online.
